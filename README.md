SQL_Myntra_Sales_Analysis

“The objective of this project was to analyze Myntra sales data and identify key factors affecting revenue and product performance.
I sourced the dataset from Kaggle, which contained raw transactional data including product categories, prices, ratings, and sales information.

I first imported the data into MySQL, where I performed data cleaning—handling missing values, removing duplicates, and standardizing formats.
Then I used SQL queries like joins, aggregations, and group by to create key performance indicators such as total revenue, average order value, and category-wise sales.

After that, I connected the processed data to Excel and built an interactive dashboard using pivot tables, charts, and slicers.
This allowed dynamic analysis of sales trends by category, price range, and rating.

From the analysis, I identified top-performing categories, pricing patterns, and how product ratings influenced sales.
This helped simulate how a business could optimize pricing and inventory decisions.

Overall, this project improved my skills in SQL, data cleaning, and dashboard creation, while also helping me understand how data-driven decisions are made.”
